PROGRAM: program { STATEMENTS_LIST }

STATEMENTS_LIST: STATEMENT | STATEMENT STATEMENTS_LIST
STATEMENT: SIMPLE_STATEMENT; | STRUCT_STATEMENT

SIMPLE_STATEMENT: DECLARATION_STATEMENT | IO_STATEMENT | ASSIGN_STATEMENT

SIMPLE_TYPE: int | long | char
ARRAY_IDENTIFIER: IDENTIFIER [ UNSIGNED ]
IDENTIFIERS: IDENTIFIER | ARRAY_IDENTIFIER
DECLARATION_STATEMENT: SIMPLE_TYPE IDENTIFIERS

ASSIGN_STATEMENT: IDENTIFIERS = EXPRESSION
EXPRESSION: EXPRESSION + TERM | EXPRESSION - TERM | TERM
TERM: TERM * FACTOR | TERM / FACTOR | TERM % FACTOR | FACTOR
FACTOR: ( EXPRESSION ) | IDENTIFIERS | CONSTANT

IO_STATEMENT: consola >> IDENTIFIERS | consola << IDENTIFIERS | consola << EXPRESSION

STRUCT_STATEMENT: PANA_STATEMENT | DACA_STATEMENT | REPETA_STATEMENT

RELATION: < | <= | > | >= | == | !=
CONDITION: EXPRESSION RELATION EXPRESSION

PANA_STATEMENT: pana CONDITION { STATEMENTS_LIST }
DACA_STATEMENT: daca CONDITION { STATEMENTS_LIST } | daca CONDITION { STATEMENTS_LIST } altfel { STATEMENTS_LIST }
REPETA_STATEMENT: repeta IDENTIFIER = EXPRESSION : EXPRESSION { STATEMENTS_LIST }
